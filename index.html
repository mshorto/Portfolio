Midnight Express

I developed the Midnight Express brand and did all of the UX and UI design work for the site. I created the logo and built the icons using Adobe Illustrator. I designed flyers and social media images for the company. I developed the site using HTML, CSS, JS, JQ and Skeleton.

Midnight Express is a business I created to help alleviate the lack of late-night transportation between San Francisco and Silicon Valley. We use crowdfunding to hire buses that run South along the Caltrain route after 2 a.m.

  • Client: Midnight Express

  • Date: December 2014

  • Service: UX/UI Design & Front-End Development

Paycrave Mobile App

Paycrave was an IOS UX/UI design project for Bloc. The assignment was to design a mobile application that allowed users to locate food trucks and seamlessly place and pay for an order.

I constructed user flows and sitemaps to start the UX process. I built wireframes using Axure RP. I designed and built all of the icons in Adobe Illustrator. My final designs were created in Sketch. I generated a high-fidelity interactive prototype using InVison.

  • Client: Bloc Project

  • Date: November 2014

  • Service: UX/UI IOS Design

Jott.ly Landing Page

Jottly was a UX/UI design project for Bloc. The assignment was to design a logo and landing page for a SaaS startup focused on allowing users to sign up, make simple content pages, and provide a way to collaborate with others within their organization.

I constructed user flows and sitemaps based on user research. I built wireframes using Axure RP. I designed the logo and built all of the icons in Adobe Illustrator. My final designs were created in Adobe Photoshop. I developed the landing page using HTML, CSS and the Skeleton framework.

  • Client: Bloc Project

  • Date: October 2014

  • Service: UX/UI Design & Front-End Development
